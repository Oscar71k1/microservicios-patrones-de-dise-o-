Sistema Universitario

Demo del sistema de gestión de inscripciones y pagos

Información del Sistema

Este es un sistema de microservicios con frontend minimalista para la gestión universitaria. Incluye autenticación, gestión de usuarios y sistema de pagos.

Gestión de Usuarios
Sistema de Pagos

Diseño Responsivo

Cómo Probar el Sistema

  1. 1
    Iniciar los Servicios
    Ejecuta npm start en la raíz del proyecto para iniciar todos los microservicios
  2. 2
    Acceder al Frontend
    Abre http://localhost:3003 en tu navegador
  3. 3
    Registrar un Usuario
    Usa el formulario de registro para crear una cuenta de prueba
  4. 4
    Explorar las Funcionalidades
    Navega por el dashboard y crea algunos pagos de prueba

Arquitectura del Sistema

  • Frontend (Puerto 3003)
    Interfaz de usuario minimalista y responsiva
  • API Gateway (Puerto 3000)
    Punto de entrada único para todas las peticiones
  • Microservicio Usuarios (Puerto 3001)
    Gestión de autenticación y perfiles de usuario
  • Microservicio Pagos (Puerto 3002)
    Gestión de órdenes de pago y transacciones

Características Destacadas

  • Diseño Minimalista
    Interfaz limpia y moderna siguiendo principios de UX/UI
  • Totalmente Responsivo
    Adaptable a dispositivos móviles, tablets y desktop
  • Autenticación Segura
    Sistema JWT con persistencia de sesión
  • SPA (Single Page Application)
    Navegación fluida sin recargas de página
